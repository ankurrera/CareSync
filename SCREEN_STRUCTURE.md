## Responsive Considerations

- **Small Screens**: Single column layout, adequate padding
- **Text Fields**: Proper keyboard types (number, text, multiline)
- **Dropdowns**: Material dropdowns with proper sizing
- **Images**: Preview scaled to fit screen width
- **Scroll**: ListView handles long content
- **Touch Targets**: Minimum 48x48 logical pixels
